### 三、rank_gap 定义

$$\text{rank\_gap} = \text{最低位次} - \text{考生位次}$$

其中 **考生位次 = 42,645**（580 分对应 2025 年一分一段表全体累计人数）。

| rank_gap 符号 | 含义 |
|---------------|------|
| **rank_gap < 0** | 院校最低位次高于考生（更靠前），考生需"向上冲"才能录取 |
| **rank_gap = 0** | 院校最低位次恰好等于考生位次，踩线 |
| **rank_gap > 0** | 院校最低位次低于考生（更靠后），考生位次有余量，录取概率较高 |

---

### 四、冲 / 稳 / 保 分档口径

| 档位 | rank_gap 范围 | 录取概率估算 | 含义说明 |
|------|---------------|-------------|----------|
| **冲** | **-15,000 ≤ rank_gap < 0** | 约 30%–60% | 考生位次在院校最低位次之上，但差距在 15,000 名以内，有一定冲刺空间 |
| **稳** | **0 ≤ rank_gap < 25,000** | 约 60%–90% | 考生位次达到或超过院校最低位次，余量在 25,000 名以内，录取把握较大 |
| **保** | **rank_gap ≥ 25,000** | 约 90%+ | 考生位次远超院校最低位次，录取把握极大，用于兜底 |
| *不推荐* | *rank_gap < -15,000* | *< 30%* | *差距过大，录取可能性极低，不纳入推荐* |

### 六、档内排序口径

每个档位内按以下规则排序：

1. **主排序**：rank_gap **由小到大**（即录取难度由高到低）
   - 冲档：rank_gap 从 -15,000 向 0 靠近，越接近 0 越有希望
   - 稳档：rank_gap 从 0 向 +25,000 递增，越接近 0 越需要关注
   - 保档：rank_gap 从 +25,000 向上递增，越接近 +25,000 越是"刚保住"

2. **同分 tie-break**：rank_gap 相同时，按 **投档计划数由大到小** 排序（招生人数多→录取机会更分散→优先）

3. **院校去重**：同一院校多个专业分别列出，不做合并，便于考生按专业偏好选择
